## Setup

### 1. Get Stripe Keys
Sign up at stripe.com and add keys to .env

### 2. Set up Email
Zoho Mail (free) or any SMTP

### 3. Deploy
```bash
git push origin main
```

## Tech Stack

- Frontend: HTML/CSS/JS (GitHub Pages)
- Backend: Node.js (optional)
- AI: Ollama / OpenAI / Anthropic
- Payments: Stripe
- Email: SendClaw
